# Deploy Frontend Jordan

## Plataforma

Cloudflare Pages.

## Build

```bash
npm run build
```

## Output

```text
.output/public
```

## Variables

```text
NUXT_PUBLIC_API_BASE=https://TU-BACKEND.onrender.com/api
NUXT_PUBLIC_APP_NAME=JORDAN
```

## Validacion

- Login carga correctamente.
- Dashboard consulta el backend.
- Pedidos, rutas, ventas y cierre diario funcionan contra Render.
